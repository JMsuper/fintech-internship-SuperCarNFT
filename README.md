## 🪄 Installation
#### - Requirement (You need to install MySQL and related modules)
```sh
pip install pymysql
pip install mysqlclient
```

(If you use Windows OS)
: Just run MySQL-client

(If you use Unix Family OS such as Linux or Mac)
```sh
sudo mysql -u root -p
```
